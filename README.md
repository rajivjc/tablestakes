# TableStakes

Practice tough negotiations against an AI with a hidden strategy, then see what you missed.

**[tablestakes.vercel.app](https://tablestakes.vercel.app)**

## How it works

1. Pick a scenario (salary, vendor pricing, client pushback) or write your own
2. Choose an opponent strategy — or let the AI pick one at random
3. Negotiate for 6 rounds with a live momentum meter
4. Get a full debrief: score, turn-by-turn analysis, strategy reveal

## Stack

Next.js · Claude Haiku · Tailwind · Vercel

## Setup

```bash
cp .env.example .env.local
# Add your ANTHROPIC_API_KEY
npm install
npm run dev
```
